
# System.plugin

> 系统插件模块

## 模块名

> systemModule

## 方法 get()

> 壳子获取手机系统信息

### 语法

> systemModule.get(options, callback);

### 参数

参数|类型|描述
-------     | -------   |--------
options     |[Object]   | 对象
callback    |[Function] | 回调函数

#### 参数 options

```
{
    app: 'APP',         // app名称
}
```

### 参数 callback

```
(event) => {}
```
