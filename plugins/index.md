
# Plugin

> APP插件模块，由壳子提供服务

## 插件模块命名规则

```
textModule

```

## 请求壳子的报文格式

```
{
    app: 'wow',
}

```

## 壳子返回的报文格式

```
{
    code: '0000',
    data: '数据'
}

```
### code的取值
> '0000' 成功
> '9999' 失败

### data的取值
> 成功时：返回成功的数据
> 失败时：返回失败的原因

## 插件模块列表

插件名|简述
---|---
resource    | 资源请求插件模块
meta        | 元数据请求插件模块
photo       | 照片插件模块
