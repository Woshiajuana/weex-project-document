
# Loading.plugin

> loading插件模块

## 模块名

> loadingModule

## 方法 show()

> 壳子显示loading动画，


### 语法

> loadingModule.show(options);

### 参数

参数|类型|描述
-------     | -------   |--------
options     |[Object]   | 对象

#### 参数 options

```
{
    app: 'APP',         // app名称
    useMask: true,      // 是否显示半透明遮罩
}
```
## 方法 hide()

> 壳子隐藏loading动画，


### 语法

> loadingModule.hide(options);
